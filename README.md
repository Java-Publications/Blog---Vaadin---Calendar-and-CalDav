# This project has moved 🚚

The active development of this calendar / CalDAV component now lives at:

## → [github.com/svenruppert/chronogrid](https://github.com/svenruppert/chronogrid)

The component has been renamed and repackaged as **ChronoGrid — A
native Vaadin Flow calendar component for Java applications**, and the
add-on is now published under the `com.svenruppert.chronogrid` Maven
namespace.

This repository remains as a historical archive of the blog-post
companion code. The last release pushed from here was
[`v00.10.00`](https://github.com/svenruppert/chronogrid/releases/tag/v00.10.00)
(also tagged at the new location); everything since lives at
[`svenruppert/chronogrid`](https://github.com/svenruppert/chronogrid).

For the blog post itself + the design rationale behind the extraction
into ChronoGrid, see
[`docs/CHRONOGRID_EXTRACTION.md`](https://github.com/svenruppert/chronogrid/blob/main/docs/CHRONOGRID_EXTRACTION.md)
at the new home.
